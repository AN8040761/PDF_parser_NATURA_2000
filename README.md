# PDF_parser_NATURA_2000
This script scraps automatically data from letters related to NATURA 2000 issued by environmental agencies in Bulgaria.
It is intended to help with the mission "Data entry for evaluation of pressure over protected habitats" organized by coalition "For the nature" on https://timeheroes.org. You can sign in for the mission from this link - http://tmhr.co/pSuPmi until September 2020 and you will receive instructions.

With the help of the script more than half of data is extracted in a csv file which then can be pasted in the spreadsheet in  google sheets. entered automatica
